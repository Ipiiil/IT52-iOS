# IT52-iOS

Мобильное приложение для сообщества **IT52**, разработанное в рамках производственной практики.

Приложение предоставляет удобный доступ к мероприятиям сообщества, позволяет просматривать информацию о событиях, следить за своим календарем мероприятий и работать с личным профилем.

---

## Цель проекта

Создать современное iOS-приложение для сообщества IT52, позволяющее пользователям:

- просматривать мероприятия;
- регистрироваться на события;
- управлять личным календарем;
- получать актуальную информацию о мероприятиях через API.

---

## Используемые технологии

- Swift
- SwiftUI
- MVVM
- URLSession
- Async/Await
- Observation
- JSONDecoder
- REST API

---

## API

Приложение использует официальный API проекта IT52.

```
https://www.it52.info/api/v2
```
